Loan Prediction

This project predicts loan approval status using machine learning models on a dataset with applicant details like income, education, and credit history.
The data is preprocessed (handling missing values, encoding, etc.), and models like Logistic Regression and Random Forest are trained.
Random Forest performs the best with around 80% accuracy, making it effective for loan prediction tasks.

Sales Prediction

This project aims to predict item sales using machine learning on a retail dataset with store and item information.
After cleaning and feature engineering, models like Linear Regression and XGBoost are trained and evaluated.
XGBoost gives the best performance, showing high accuracy in predicting future item sales..
